â“ FAQ & TROUBLESHOOTING

Q: Bagaimana jika admin salah input NISN?
A: Admin dapat edit siswa dan ganti NISN. Password login akan otomatis update.

Q: Bagaimana jika siswa lupa NISN?
A: Admin dapat lihat NISN di tabel daftar murid dan komunikasikan ke siswa.

Q: Apakah masih bisa login pakai email?
A: Tidak, siswa harus menggunakan NISN. Email hanya untuk komunikasi admin.

Q: Apa bedanya password siswa?
A: Password siswa sama dengan NISN-nya. Admin sangat menyarankan siswa untuk 
   mengubah password saat first login (feature ini bisa ditambah di iterasi berikutnya).

Q: Bagaimana menghapus siswa?
A: Di admin panel, klik "Hapus" pada baris siswa. Akun siswa juga otomatis 
   terhapus dari sistem.

Q: Apakah NISN harus 10 digit?
A: NISN resmi memang 10 digit numerik, tapi sistem ini bisa menerima string 
   apa saja (belum ada validasi format NISN).

Q: Bisa ganti NISN setelah siswa terdaftar?
A: Ya, admin bisa edit siswa dan ganti NISN. Siswa harus login pakai NISN baru.
